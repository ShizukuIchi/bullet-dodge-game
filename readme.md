## Game Guide

### Character

- Left click - Change character direction
- Left long press - Charge Energy (Three times for ultimate)
- Right click - Taunt

### Items

- Dodge every kind of bullets
- Items may either buff or debuff the character

### Map

- It will rotate randomly
